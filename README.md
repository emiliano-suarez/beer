# beer
Our beloved beer, site

## Install app

```sh
$ cd /home/httpd/sites/
git clone git@github.com:emiliano-suarez/beer.git
$ cd beer
$ chmod 777 -R storage
$ composer install
$ php artisan cache:clear
```
